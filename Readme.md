# Rust Training Repository

========================
This repository contains a collection of Rust programming exercises and examples designed to help learners improve their Rust skills. Whether you're a beginner or an experienced developer, you'll find valuable resources here to enhance your understanding of Rust.

## Resources Links

- [The Bible: The Rust Book](https://doc.rust-lang.org/)

## Exercises

1. [Guessing Game](./guessing_game/Readme.md) : After Chapter 2 of The Rust Book.
2. [PNG Metadata Parser](./png_metadata_parser/Readme.md) : After Chapter 10 of The Rust Book.
3. [Custom Grep Tool](./custom_grep_tool/Readme.md) : After Chapter 12 of The Rust Book.
4. [Static Site Generator](./static_site_generator/Readme.md) : After Chapter 13 of The Rust Book.
5. [Custom Shell](./custom_shell/Readme.md) : After Chapter 12 of The Rust Book and documentation of STD::process module.
6. [A Redis-lite Key-Value Store](./redis_lite/Readme.md) : After Chapter 16 of The Rust Book.
7. [A Single-threaded BitTorrent Client](./single_threaded_bit_torrent_client/Readme.md) : After Chapter 17 of The Rust Book.
